D2MS

D2MS

한국군 표준군대부호(MND-STD-2525C) 처리를 위한 기본클래스로 기본부호인 전술기호(Tactical Symbol)와 전술도식(Tactical Grapphic) 정형 부호를 표출하는 전용 클래스이다.
작전활동부호인 전술도식(Tactical Graphics) 중 선형과 면형 객체는 MSTacticalLineGraphics과 MSTacticalPolygonGraphics로 각각 관리되고 있다.

Constructor

new D2MS()

Methods

asCanvas()

군대부호 처리 결과를 이미지 형식으로 반환

Example
var image = sym.asCanvas().toDataURL('image/png', 1);
var img = document.createElement("img"); 
img.src = imgSrc;
imgContainer.append(img);

asSVG()

군대부호 처리 결과를 SVG 형식으로 반환한다. SVG는 벡터 형식이므로 asCanvas()로 생성된 이미지보다는 부호의 선명도가 훨씬 좋다.

Example
var imgSrc = "data:image/svg+xml;charset=utf-8," + encodeURIComponent(sym.asSVG());
var img = document.createElement("img"); 
img.src = imgSrc;
imgContainer.append(img);

setOptions(options)

군대부호 전시를 위한 기본정보를 설정한다.

Example
//D2MS 군대부호 객체 생성
var sym = new D2MS.ms.Symbol('');
//속성정보 정의
var options = {
     SIDC : 'SFP*S-----*****',
     icon : true,
     frame : true,
     fill : true,
     size : 14
};

//[참고사항]
//군대부호 크기(가로 x 세로) 확인 방법
console.log(sym.getSize()); //군대부호 이미지(픽셀) 크기 조사

//[참고사항]
//군대부호 객체가 선택되어진 경우를 표시할 때 군대부호 외곽선색상과 두께로 선택 상태 제어 가능
//선택이 해제될 경우에는 이전 값으로 복구해 주어야 함 
//군대부호 외곽선 색상 설정 방법
//options.outlineColor = "#FFAF3A";
//군대부호 외곽선 두께 설정 방법
//options.outlineWidth = "10";

//속성정보 설정
sym.setOptions(options);
//SVG 방식으로 군대부호 전시
var imgSrc = "data:image/svg+xml;charset=utf-8," + encodeURIComponent(sym.asSVG());
//이미지 표시용 컨테이너 할당
var imgContainer = $("#milsymbol-sub-explorer-img"); 
//이미지 요소 생성
var img = document.createElement("img"); 
img.src = imgSrc;
//기존 이미지 삭제
imgContainer.empty(); 
//이미지 컨테이너에 이미지 전시
imgContainer.append(img);
Parameters:
Name Type Description
options object

기본정보 설정

  • SIDC{string} : 군대부호 코드값(15자리)
    . ex → SIDC : “SGUULS--*****”
  • icon{bool} : 기능부호 표시
    . ex → icon : true
  • frame{bool} : 외형부호 표시
    . ex → frame : true
  • fill{bool} : 외형채움 표시
    . ex → fill : true
  • civilianColor{bool} : 민간부호 채움색 설정
  • fillOpacity{number} : 투명도(Transparency) 설정(0.0 ~ 1.0)
    . ex → fillOpacity : 1.0
  • operationalConditionPoint{number} : 상태(15자리 군대부호 코드 중 4번째 코드값) 대한 운용조건 표시(0 : 미표시, 1 : 중앙, 2 : 하단)
    . ex → operationalConditionPoint : 2
  • userDefineLineColor{color} : 점형 작전활동부호 피아 선색상을 사용자 정의 색상으로 설정(RGB 혹은 RGBA)
    . ex → userDefineLineColor : "rgb(255,0,255)"
  • userDefineFillColor{color} : 피아 배경색을 사용자 정의 색상으로 설정(RGB 혹은 RGBA)
    . ex → userDefineFillColor : "rgb(255,255,0)"
  • affiliation{string} : 피아구분(군대부호 2번째 자리)
  • status{string} : 상태(군대부호 4번째 자리)
  • size{number} : 심볼 크기(3 ~ 14)
    . ex → size : 7
  • strokeWidth{number} : 선 두께
  • infoColor{color} : 수식정보 색상(평가등급코드, 전투력, 적군표시, IFF/SIF을 제외한 나머지에 적용, RGB 혹은 RGBA)
  • infoFields{bool} : 수식정보 표시여부
  • colorMode{string} : 부호 밝기 조절(Light 혹은 Medium 혹은 Dark로 설정)
    . ex → colorMode : "Dark"
  • userDefineBrightness{string} : 부호 밝기 조절(0 ~ 100사이의 % 값으로 설정)
  • addSymbol{string} : 부호 내부에 심볼 추가 표시(작전활동부호 점형만 해당)
    . ex → addSymbol : "GMOMP---****X"
  • addSymbolOnly{bool} : 외곽선 없이 내부 심볼만 표시(작전활동부호 점형만 해당)
  • uniqueDesignation{string} : T : 고유명칭
    . ex → uniqueDesignation : "T"
  • uniqueDesignation1{string} : T1 : 고유명칭1(작전활동부호에서 적용)
  • additionalInformation{string} : H : 활동사항
  • additionalInformation1{string} : H1 : 활동사항1(작전활동부호에서 적용)
  • dtg{string} : W : 활동시각
  • dtg1{string} : W1 : 활동시각1(작전활동부호에서 적용)
  • hostile{string} : N : 적군표시
  • quantity{string} : C : 수량
  • location{string} : Y : 위치
  • type{string} : V : 장비명/종류
  • altitudeDepth{string} : X : 고도/심도/거리
  • altitudeDepth1{string} : X1 : 고도/심도/거리(작전활동부호에서 적용)
  • mobileUnit{bool} : D : 기동부대 표시
  • signatureEquipment{string} : L : 신호정보장비
  • direction{string} : Q : 이동방향각도(자북기준시계방향)
  • speed{number} : Z : 속도
  • reinforcedReduced{string} : F : 부대증감
  • staffComments{string} : G : 군/국가구분코드
  • higherFormation{string} : M : 상급부대
  • headquarters{bool} : S : 지휘소/실제위치 표시
  • showCombatEffectivenessLabel{bool} : 전투력 레이블 표시 여부
  • iffSif{string} : P : 수식정보(showCombatEffectivenessLabel : true 일 경우 적용)
  • mobilityIndicator{string} : R : 기동수단 코드(O, P, Q, R, S, T, U, V, W, X, Y 중 설정)
  • mobilityCode{string} : R2 : 이동성 코드(M 혹은 S 혹은 U로 설정, M은 이동가능, S는 고정, U는 불확실을 의미)
  • evaluationRating{string} : J : 평가등급코드(showCombatEffectivenessLabel)가 true인 경우 확인 가능)
  • combatEffectiveness{number} : K : 전투력(showCombatEffectivenessLabel가 true인 경우 확인 가능), 0 ~ 100 값으로 설정, 녹색(100 ~ 90), 노란색(89 ~ 75), 빨간색(74 ~ 50), 검정색(49 ~ 0)으로 관리)
  • showCombatEffectivenessWaterFill{bool} : 전투력 물채움 표시
  • useDefineColorTocombatEffectiveness{color} : 사용자 정의 색상으로 물채움 설정(RGB 혹은 RGBA)
  • specialHeadquarters{string} : AA : 지휘통제소명
  • feintDummyIndicator{string} : AB : 가장/가상식별부호(1로 설정시 우산모양표시)
  • platformType{string} : AD :기반형태(ELNOT 혹은 CENOT로 설정)
    . ex → platformType : "ELNOT"
  • equipmentTeardownTime{string} : AE : 분해시간
  • commonIdentifier{string} : AF : 공통명칭
  • auxiliaryEquipmentIndicator{string} : AG : 보조장비코드(shortSonar 혹은 longSonar로 설정)
