Distance

거리 측정시 표시여부 상태값을 반환한다.

Returns:

Type
bool

Methods

add(mode)

거리 혹은 경사 거리를 측정한다.

Example
window.map._distance.setUnit('meter');              //단위(meter, mile, nauticalmile)
window.map._distance.setSpeed(60);                  //속도(km/h)
window.map._distance.setLineWidth(3);               //선두께
window.map._distance.setLineColor([255, 255, 0, 0.8]);  //선색상
window.map._distance.setLineVisible(true);           //선표시 여부
window.map._distance.setTextVisible(true);           //텍스트 표시여부
window.map._distance.setSegmentTextVisible(true);    //구간거리 표시 여부
window.map._distance.setScaleByDistance(false);      //빌보드 크기 자동조절
window.map._distance.setHeightOffset(1000);          //빌보드 높이값 조절
window.map._distance.setBearing(true);               //방위각 표시
window.map._distance.add('distance');                 //거리측정 추가
Parameters:
Name Type Description
mode string

모드 설정(distance, slope 중 택일)

setSpeed(speed)

속도(km/h)를 설정한다. 설정한 속도를 기준으로 구간 소요 시간이 계산되어 표시된다.
속도가 0이면 소요시간은 표시되지 않는다.

Example
window.map._distance.setSpeed(60); //km/h

Name Type Description
speed number

속도를 입력한다.
