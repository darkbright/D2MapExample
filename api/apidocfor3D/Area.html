Area

Area

사용자가 설정한 영역의 면적을 측정하기 위한 클래스로 측정 단위(meter, mile)에 따라 결과가 표시된다.

Constructor

new Area()

Methods

add()

면적을 측정한다.

Example
window.map._area.setScaleByDistance(false);         //빌보드 크기 자동조절
window.map._area.setHeightOffset(1000);             //빌보드 높이값 조절
window.map._area.setAreaColor([255, 255, 0, 0.8]);  //면적색상
window.map._area.setAreaVisible(false);             //면적표시 여부
window.map._area.setTextVisible(true);              //텍스트 표시여부
window.map._area.add();

setAreaColor(val)

면적측정 색상을 설정한다.
처음부터 색상을 사용자가 정의한 색상으로 설정할 경우에는 map._area.add()를 생성하기 전에 호출해야 한다.

Example
window.map._area.setAreaColor([255, 255, 0, 0.8]); //면적측정 색상 설정

Name Type Description
