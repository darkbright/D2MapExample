EventManager

EventManager

지도/지형분석/투명도 등 서비스 모드를 관리하는 클래스이다.

Constructor

new EventManager()

객체 초기 상태를 설정한다.

Methods

(static) getDoubleClick()

마우스 더블 클릭이 되었는지 결과를 리턴한다.
더블 클릭이 되었을 경우 true 값을 리턴한다.

(static) getMapMode()

현재 서비스 모드를 리턴하며 모드 값은 다음과 같다.

  • default
  • measurement
  • terrainAnalysis
  • grahic

(static) setMapMode(menu)

서비스별 모드를 설정한다.

Parameters:
Name Type Description
menu string

서비스 모드 설정

  • default : 사용자 정의 모드 설정 시 적용되며 측정기능/지형분석/투명도는 비활성화 처리
  • terrainAnalysis : 측정관리와 지형분석 설정 시 적용
  • graphic : 투명도 작성 및 편집 설정 시 적용되며 측정기능/지형분석 서비스는 비활성화 처리
  • cop : 사용자 정의 모드로 상황도 구성 및 설정시 지도/투명도 관련 기능은 모드 비활성화 처리
